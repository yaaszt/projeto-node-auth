# Projeto Node – Sistema de Login e Cadastro

Este projeto foi desenvolvido com auxílio do ChatGPT, utilizando o mesmo prompt aplicado na plataforma Lovable, com o objetivo de comparar os resultados entre diferentes ferramentas de Inteligência Artificial.

## Objetivo

Desenvolver uma aplicação web contendo:

- Página de cadastro de usuários  
- Página de login  
- Backend em Node.js  
- Banco de dados SQLite  
- API REST para autenticação  

## Tecnologias Utilizadas

- Node.js  
- Express  
- SQLite  
- HTML  
- CSS  
- JavaScript  

## Execução Local

1. Clone o repositório:  
   git clone https://github.com/yaaszt/projeto-node-auth.git

2. Acesse a pasta do projeto:  
   cd projeto-node-auth

3. Instale as dependências:  
   npm install

4. Inicie o servidor:  
   npm start

5. Acesse no navegador:  
   http://localhost:3000

## Estrutura do Projeto

O projeto possui separação entre:

- Servidor (server.js)  
- Banco de dados (database.js)  
- Arquivos públicos (HTML, CSS e JavaScript)  

## Observação

Diferentemente do projeto desenvolvido com Lovable, neste caso o backend e o banco de dados são configurados manualmente e executados localmente, demonstrando uma abordagem tradicional de desenvolvimento com Node.js e SQLite.
